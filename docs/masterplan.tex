\section{Project Overview}

The \textbf{Workflow \& Approval Management System (WAMS)} is an industry-oriented backend system designed to manage multi-step workflows that require role-based approvals and controlled state transitions.

This project is used as the primary vehicle to learn Java, Spring Boot, and backend engineering fundamentals in a structured, production-aligned manner.

---

\section{Learning Philosophy}

The execution plan follows these principles:

\begin{itemize}
  \item Fundamentals before complexity
  \item Monolith first, then evolution
  \item One concept at a time
  \item Each phase ends with working code and Git commits
  \item Buzzwords are introduced only when justified
\end{itemize}

---
